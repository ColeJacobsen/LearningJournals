## Content Insights:
The HTML language is different from most programming languages I know. It seems simple on the surface but the variety of tags that can be applied to each element allows for vast complexity. 
Such tags include:
h1 - h6 for the various size of headers
small - for small text
button - for a variety of buttons when combined with different classes
Said classes also allow for customization of text, such as swapping the color with: class="text-primary"
Basic colors also have a name associated with them. Primary = blue, Success = green, Danger = red, etc. 
There are also a variety of utility classes that help adjust things like:
m or p(y, x, t, b) - margin and padding position
border-(top, end, start, bottom) - border position
shadow-(sm, lg) - box shadow 
HTML also works in a grid system, so every container involved can be placed in a specific row/column.
