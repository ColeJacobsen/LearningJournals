## Content Insights:
WordPress themes are a collection of files that change the default visuals of a WordPress site. This allows for changes to be made to background colors, text fonts, image placements, and more. Child themes use another theme as a base, then modify that by overriding some of the inherited traits. This is useful for light modifications, such as palette swaps. Child themes make these changes via a file named style.css. Child themes also can not become parent themes. 

## Career/Employability/Learning Insights:
Since WordPress is the most used CMS on the market, knowing how to use it further is always helpful. Additionally, understanding how to use child themes will allow for an easier time when designing websites. Since parent themes do most of the work, a child theme is a great way to make something quick and personal.
