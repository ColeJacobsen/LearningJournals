## Content Insights:
This week was focused on building child themes in WordPress
The public file in a WordPress site contains all the plugins and themes for a site. 
In wp-content/themes, create a new folder to make a new theme. 
The new theme must have a style.css file, which should have the basic content from the offical WordPress child theme site copy and pasted. 
Changes to the existing parts of the parent theme style.css can be made via the child theme's style.css file as it overwrites its inheritence. These cnages are typically written in php. 
Using the inspect feature found in most browsers, you can accurately identify what parts you want to change.
